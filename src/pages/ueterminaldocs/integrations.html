Integrations

AI tools and command-line utilities compatible with Octree Terminal

AI Integration

Octree Terminal is designed to work seamlessly with any AI tool that operates through a standard command-line interface. Because Terminal provides a fully functional shell environment inside Unreal Engine, any CLI-based AI assistant can run directly within the editor.

Compatible Tools

The following AI tools and CLI utilities have been tested and confirmed to work inside Octree Terminal:

Tool

Description

unreal-mcp

MCP server for Unreal Engine integration with AI assistants

Claude Code

Anthropic’s CLI coding assistant for terminal-based AI development

Kimi CLI

Moonshot AI’s command-line interface for AI-powered coding

Crush

AI-powered CLI tool for code generation and editing

Factory Droid

Automated AI agent for code refactoring and development tasks

Gemini CLI

Google’s command-line interface for Gemini AI models

GitHub Copilot CLI

GitHub’s AI pair programmer for terminal command suggestions

OpenAI Codex

OpenAI’s CLI agent for code generation and software engineering

Opencode

Open-source terminal-based AI coding assistant

Any CLI tool that runs in a standard terminal can run inside Octree Terminal. If your preferred AI tool has a command-line interface, it will work out of the box — no special configuration needed.
