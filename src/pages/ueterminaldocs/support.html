Support Channels

If you need help or want to report an issue, you can reach us through any of the following channels:

Fab Marketplace

Submit support requests and questions through the Fab Marketplace product page.

Discord Community

Join our Discord server for real-time help, discussions, and community support.

Bug Reports

Found a bug? Report it through the Fab Marketplace or Discord with steps to reproduce.

Reporting Bugs

When reporting a bug, please include the following information to help us resolve the issue quickly:

Unreal Engine version (e.g. 5.4, 5.5)

Windows version (e.g. Windows 10, Windows 11)

Steps to reproduce the issue

Expected behavior vs. actual behavior

Any relevant error messages or logs
